AllWize - Pollution monitoring

This example use cases uses a MICS-4514 sensor to retrieve CO and NO2 values.

If you are using the AllWize K1 you must connect the grove cable to the analog port (the middle one). Connect the yellow cable leading to that port (A0) to the NOX pin and the white cable to the RED pin.

You will also need a simple dupond cable between D8 and the PRE pin in the sensor, this pin us used to trigger the pre-heat process.

The sensor is a 5V sensor so ensure you have the power jumper properly set.

Always remember to connect the antenna before powering the board!
